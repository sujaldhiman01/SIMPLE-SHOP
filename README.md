🛍️ Simple Shop Web App

A minimal and responsive shopping cart web application built using HTML, CSS, and JavaScript. This project demonstrates core front-end concepts like dynamic rendering, state management, and interactive UI without any external libraries.

🚀 Features
🛒 Product listing with categories and prices
➕ Add items to cart
🔄 Increase / decrease item quantity
❌ Remove items automatically when quantity reaches zero
💰 Dynamic total price calculation
🚚 Conditional shipping:
Free shipping on orders above $100
$12 shipping otherwise
📦 Real-time cart updates
📱 Fully responsive layout

🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Logic and interactivity

📂 Project Structure
simple-shop/
│── index.html   # Main application file (HTML + CSS + JS)
│── README.md    # Project documentation
⚙️ How It Works
Products are stored in a JavaScript array (ITEMS)
Cart data is managed using an object (cart)
UI updates dynamically using the render() function
Quantity controls update cart state in real-time
